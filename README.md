# 字幕快速清除工具

使用 Python + OpenCV 的可视化小工具，可批量载入视频，逐个视频框选字幕区域并使用高斯模糊处理。支持拖动时间轴预览、逐个视频独立编辑，并在本地尝试保留音频（需要本机安装 `ffmpeg`）。

## 环境准备

1. 建议 Python 3.9+。
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 可选：安装 `ffmpeg` 以在导出时自动合并原始音轨。未安装 `ffmpeg` 时仍可导出视频，但会丢失音频。

## 运行

```bash
python main.py
```

## 使用步骤

1. 点击 “添加视频” 选择一个或多个视频文件。
2. 在右侧列表选择要编辑的视频。
3. 拖动下方时间轴选择包含字幕的帧，鼠标在预览画面中拖拽框选字幕区域（可随时清除重新框选）。
4. 调节 “模糊强度”（数值越大模糊越强）。
5. 如需导出时去除音轨，勾选 “去除音轨”；默认会尝试保留原音频（需要 `ffmpeg`）。
6. 默认会自动检测视频比例，若不是 9:16 会裁剪到 9:16（从中心裁剪，无黑边）；可取消勾选 “自动 9:16 裁剪” 以保持原比例。
7. 若需自定义导出目录，点击 “选择导出目录”；未选择时默认导出到源文件所在目录。
8. 勾选 “导出后从列表移除” 可在导出完成后自动从程序列表移除对应视频（不影响源文件）。
9. 未框选字幕区域时也可直接导出（仅应用裁剪/音频/比例设置）。
10. 编码器可选 `自动检测`（默认，优先 NVENC / QSV / AMF，失败自动回退软件 x264）、`h264_nvenc`、`h264_qsv`、`h264_amf`、`libx264`、`copy`（需 ffmpeg 支持对应硬件编码）。
11. 点击 “开始处理” 导出列表中的视频。输出文件名称追加 `_blurred` 后缀。
12. 如果自动检测不到 ffmpeg，可在导出选项里点击 “浏览” 手动指定可执行文件。

## 已知说明

- 输出默认使用 OpenCV 写入 MP4；若本机未安装 `ffmpeg`，导出的视频将没有音频。
- 导出会通过 `ffprobe` 读取源视频码率，并以目标码率 + maxrate/bufsize 的方式尽量保持原有码率；无法读取时回退到质量优先（CRF/CQ）模式。
- 充分利用显卡：解码优先走 CUDA / D3D11VA / DXVA2 / VAAPI 硬件加速，模糊在可用时使用 OpenCV CUDA 滤镜，编码优先使用 NVENC / QSV / AMF 硬件编码器。
- 框选区域与模糊设置在同一视频内复用，可在时间轴任意帧预览模糊效果。
- GUI 会根据当前屏幕 DPI 自动缩放，适配高分屏。
- 右侧“日志”面板可查看处理进度与 ffmpeg 回退情况。（日志位于列表下方）
- 单窗口布局：左侧为预览与设置，右侧为列表与日志，分隔条可拖动调整比例。

## 代码结构

- `main.py`：图形界面与交互逻辑。
- `app_config.py`：配置文件的读写与旧版本迁移（`AppConfig`）。
- `media_tools.py`：ffmpeg / ffprobe 探测、硬件解码/编码检测、源码率读取、编码参数生成。
- `exporter.py`：`VideoExporter` 导出管线（解码 → ROI 模糊 → 可选 9:16 裁剪 → 编码），含硬件→软件编码回退。
- `blur_core.py`：模糊算法实现，支持 OpenCV CUDA 加速。
- `perf_benchmark.py`：模糊算法的 CPU 性能基准脚本。
